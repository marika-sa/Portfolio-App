An updated-modernised version of my personal portfolio site.

## Introduction
Hi there, thank you for checking out my project! This is a Next.js web app I created to showcase my projects, experiences and share my journey of becoming a developer.


## About this app
This app was fully created by, starting with the design process making wireframes and a higher fidelty version in **Figma**. Moving onto the code, it's a project created with **Next.js** and **TypeScript**. 

## To run the app
First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Then open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
